# Relatório de Análise de Vendas — Hermann’s Cake and Coffee Shop

Projeto prático desenvolvido na disciplina Introdução à Visualização de Dados e SQL, com foco em análise e comunicação de dados. 
O objetivo foi criar um relatório visual e interativo utilizando o **Google Looker Studio**, a partir de uma base de dados em formato `.xlsx`.

A proposta do trabalho foi transformar dados brutos de vendas em informações claras e fáceis de interpretar.

Relatório completo em PDF disponível neste repositório.

---

## Objetivo do Projeto

Analisar as vendas da cafeteria **Hermann’s Cake and Coffee Shop**, buscando responder perguntas como:
- Quais produtos mais vendem?
- Quais categorias têm maior participação nas vendas?
- Qual o volume total de itens vendidos?

O foco esteve na **organização da informação e clareza visual**, facilitando a leitura e a análise dos dados.

---

## Fonte dos Dados

Os dados utilizados estão no arquivo:

- `DS - Coffee Shop.xlsx`

O dataset contém:
- Nome do produto  
- Descrição  
- Sabor  
- Tipo de produto (Cupcake, Cake, Loaf, Cookie)  
- Quantidade vendida  

---

## Estrutura do Relatório

O relatório foi dividido em duas partes principais:

### Tabela de Detalhes do Produto
Uma tabela detalhada exibindo as características de cada item vendido, permitindo:
- Comparação entre produtos
- Identificação de destaques, como o **Lunch Bun**, responsável por 10.000 unidades vendidas
- Análise de produtos com baixa saída, como o **Sicilian Lemon Cake**

### Dashboard Resumo
Um painel visual contendo:
- **Scorecard** com o total de unidades vendidas (**14.123 itens**)
- **Gráficos de pizza** mostrando a participação por categoria:
  - Cupcakes: 40% das vendas
  - Cakes: 30% das vendas
  - Demais categorias distribuídas no restante
